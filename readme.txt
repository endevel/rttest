Тестовое задание в Ростелеком Информационные Технологии
========================================================

1. Реализация класса в файле CircleBuffer
2. Классы Customer и Producer сделаны исключительно для тестовых прогонов и трассировки работы ( пока без логгиования )
3. Для тестовый прогонов используем класс AppTestRunner с выводом сообщений в консоль
4. Unit-тесты собраны в классе CircleBufferTest (NGTest)
